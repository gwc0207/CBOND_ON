# 因子解释手册（当前 Live 版本）

## 1. 口径与来源
- 本文只覆盖当前 `live` 实际启用因子（共 27 个）。
- 因子配置来源：`cbond_on/cbond_on/config/live/live_factors_config.json5`
- 因子实现来源：`cbond_on/cbond_on/domain/factors/defs/*.py`

## 2. 通用计算约定
- 面板按 `(dt, code, seq)` 组织，因子输出为 `(dt, code)` 一条值。
- `daily_*` 因子依赖 `market_cbond.daily_twap` 日频表，再回填到面板日 `dt`。
- 对 `alpha` 系列（继承 `_AlphaBase`）：
  - 最终会执行 `replace([inf, -inf], NaN).fillna(0.0)`。
  - 仅当参数里显式传 `windowsize`（或 `window_size`）时，触发 OHLC 重构。

### 3.1 日频隔夜收益均值（5 个）

1) `cb_overnight_return_mean_20d`  
- 实现：`daily_overnight_return_mean_v1`  
- 参数：`buy_col=twap_1442_1457, sell_col=twap_0930_1000, window=20`  
- 公式：  
  - `prev_buy_t = buy_{t-1}`  
  - `overnight_ret_t = sell_t / prev_buy_t - 1`（要求两者 > 0）  
  - `rolling_mean_t = MA_window(overnight_ret)_t`  
  - `factor_t = rolling_mean_{t-1}`（代码里有 1 日 shift）

### 3.2 日频隔夜 Sharpe（4 个）

  - `overnight_ret_t = sell_t / prev_buy_t - 1`

### 3.3 盘中基础因子（8 个）

10) `range_30m`  
- 实现：`range_ratio`  
- 参数：`window_minutes=30, price_col=last`  
- 公式：`factor = (max(last)-min(last)) / last_end`（窗口内）

11) `amount_30m`  
- 实现：`amount_sum`  
- 参数：`window_minutes=30, amount_col=amount`  
- 公式：`factor = sum(amount)`（窗口内）

12) `vol_30m`  
- 实现：`volatility`  
- 参数：`window_minutes=30, price_col=last, use_log_return=true`  
- 公式：`ret_i = log(last_i) - log(last_{i-1})`，`factor = std(ret)`

13) `volume_30m`  
- 实现：`volume_sum`  
- 参数：`window_minutes=30, volume_col=volume`  
- 公式：`factor = sum(volume)`（窗口内）

14) `mid_move_30m`  
- 实现：`midprice_move`  
- 参数：`window_minutes=30, bid_col=bid_price1, ask_col=ask_price1`  
- 公式：`mid=(bid1+ask1)/2`，`factor=(mid_end-mid_start)/mid_start`

15) `mom_slope_30m`  
- 实现：`mom_slope`  
- 参数：`window_minutes=30, price_col=last`  
- 公式：窗口内对 `price ~ time_seconds` 做一元线性回归，取斜率 `beta`

16) `ret_10m`  
- 实现：`ret_window`  
- 参数：`window_minutes=10, price_col=last`  
- 公式：`factor=(last_end-last_start)/last_start`

17) `depth_weighted_imbalance_v1`  
- 实现：`depth_weighted_imbalance_v1`  
- 参数：`weights=[5,4,3,2,1]`  
- 公式：  
  - `bid_w = Σ(w_i * bid_volume_i)`  
  - `ask_w = Σ(w_i * ask_volume_i)`  
  - `factor = (bid_w - ask_w) / (bid_w + ask_w + 1e-8)`  
  - 使用当日末条快照值（每个 `(dt,code)` 的最后一条）

### 3.4 股债联动与盘口结构（2 个）

18) `premium_momentum_proxy_v1`  
- 实现：`premium_momentum_proxy_v1`  
- 依赖：`ctx.panel + ctx.stock_panel + ctx.bond_stock_map`  
- 公式：  
  - `bond_strength = (bond_last - bond_prev_bar_close)/bond_prev_bar_close`  
  - `stock_strength = (stock_last - stock_prev_bar_close)/stock_prev_bar_close`  
  - `factor = bond_strength - stock_strength`

19) `volen_f60_s10_l3`  
- 实现：`volen`  
- 参数：`fast=60, slow=10, levels=3`  
- 公式：  
  - 每条快照：`total_vol = Σ_{i=1..3}(ask_volume_i + bid_volume_i)`  
  - 取组内尾部 `fast` 条均值与尾部 `slow` 条均值之比  
  - `factor = mean_fast(total_vol) / mean_slow(total_vol)`

### 3.5 Alpha 因子（8 个）

20) `alpha001_signed_power_v1`  
- 实现：`alpha001_signed_power_v1`  
- 参数：`stddev_window=20, ts_max_window=5, windowsize=10`  
- 公式：  
  - `ret=(last-prev_bar_close)/(prev_bar_close+eps)`  
  - `std_ret=rolling_std(ret, stddev_window)`  
  - `base = std_ret (ret<0) else last`  
  - `sp = sign(base) * |base|^2`  
  - `raw = rolling_max(sp, ts_max_window)_last`  
  - `factor = cs_rank(raw) - 0.5`

21) `alpha030_close_sign_volume_v1`  
- 实现：`alpha030_close_sign_volume_v1`  
- 参数：`delay1=1, delay2=2, delay3=3, sum_window_short=5, sum_window_long=10`  
- 公式：  
  - `S = sign(last_t-last_{t-d1}) + sign(last_{t-d1}-last_{t-d2}) + sign(last_{t-d2}-last_{t-d3})`  
  - `R = 1 - cs_rank(S)`  
  - `V = rolling_sum(volume, short) / (rolling_sum(volume, long)+eps)`  
  - `factor = R * V`

22) `alpha024_close_trend_filter_v1`  
- 实现：`alpha024_close_trend_filter_v1`  
- 参数：`sum_window=20, delta_window=20, ts_min_window=20, short_delta_window=3, trend_threshold=0.05`  
- 公式：  
  - `avg_close = rolling_mean(last, sum_window)`  
  - `rate = diff(avg_close, delta_window) / shift(avg_close, delta_window)`  
  - 若 `rate <= threshold`：`factor = -(last - rolling_min(last, ts_min_window))`  
  - 否则：`factor = -delta(last, short_delta_window)`

23) `alpha041_geometric_mean_vwap_v1`  
- 实现：`alpha041_geometric_mean_vwap_v1`  
- 参数：`windowsize=10`  
- 公式：  
  - `geo = sqrt(high * low)`  
  - `vwap = amount / (volume + eps)`  
  - `factor = geo_last - vwap_last`

24) `alpha078_low_vwap_adv_corr_v1`  
- 实现：`alpha078_low_vwap_adv_corr_v1`  
- 参数：`weight=0.352, sum_window_1=20, adv_window=20, sum_window_2=20, corr_window_1=7, corr_window_2=6, windowsize=10`  
- 公式：  
  - `vwap = amount/(volume+eps)`  
  - `weighted = weight*low + (1-weight)*vwap`  
  - `sum_weighted = rolling_sum(weighted, sum_window_1)`  
  - `adv = rolling_mean(amount, adv_window)`  
  - `sum_adv = rolling_sum(adv, sum_window_2)`  
  - `r1 = rank_pct(corr(sum_weighted, sum_adv, corr_window_1))_last`  
  - `r2 = rank_pct(corr(rank_pct(vwap), rank_pct(volume), corr_window_2))_last`  
  - `factor = r1 ^ r2`

25) `alpha019_close_momentum_sign_v1`  
- 实现：`alpha019_close_momentum_sign_v1`  
- 参数：`delta_window=7, sum_window=250, windowsize=10`  
- 公式：  
  - `sign_term = sign((last_t-last_{t-d}) + diff(last,d)_t)`  
  - `sum_ret = rolling_sum((last-prev_bar_close)/prev_bar_close, sum_window)_last`  
  - `factor = (-sign_term) * (1 + cs_rank(1 + sum_ret))`

26) `alpha025_return_volume_vwap_range_v1`  
- 实现：`alpha025_return_volume_vwap_range_v1`  
- 参数：`adv_window=10, windowsize=10`  
- 公式：  
  - `ret = (last-prev_bar_close)/(prev_bar_close+eps)`  
  - `adv = rolling_mean(amount, adv_window)_last`  
  - `vwap = amount_last/(volume_last+eps)`  
  - `price_range = high_last - last_last`  
  - `raw = (-ret) * adv * vwap * price_range`  
  - `factor = cs_rank(raw)`

27) `alpha050_volume_vwap_corr_max_v1`  
- 实现：`alpha050_volume_vwap_corr_max_v1`  
- 参数：`corr_window=5, ts_max_window=5, windowsize=10`  
- 公式：  
  - `vwap = amount/(volume+eps)`  
  - `rank_vol = rank_pct(volume)`，`rank_vwap = rank_pct(vwap)`  
  - `corr = rolling_corr(rank_vol, rank_vwap, corr_window)`  
  - `rank_corr = rank_pct(corr)`  
  - `factor = -rolling_max(rank_corr, ts_max_window)_last`
